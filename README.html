:: Readme :: Looks 1.2.1 ::

Free high-fidelity Windows and multi-platform appearance

The JGoodies Looks make your Swing applications and applets look better. The package consists of a Windows look&feel and the Plastic look&feel family. These have been optimized for readability, precise micro-design and usability.

The Looks require Java 1.4 or later. A version for Java 1.3 is here.

Getting Started

First, download and run the JGoodies Looks Demo. Then follow the Quick Start and read the user's guide. If you want to use a JGoodies l&f in NetBeans read here. As a next step you can study the JGoodies articles and presentations.

Support

You can subscribe to the users mailing list and browse or search the list's archive. In case of a new problem, you can direct questions to users@looks.dev.java.net. Also, you can use the issue tracker to find and enter issues around the Looks. Do not send personal mail!

Related Projects

The winlaf project hosts another Windows look&feel. Recently we started to join our efforts in making the Windows look&feel look and feel like Windows.

As of the Looks 1.2.1 and winlaf 0.4.1, the JGoodies Windows l&f provides a more precise micro design for 96dpi and 120dpi which leads to better readability and legibility if used with a precise layout system such as the JGoodies Forms. Also, the JGoodies Windows l&f uses component sizes that are closer to the native original and make it easier to switch the look&feel to the multiplatform Plastic l&f family.

Currently both the winlaf and JGoodies l&f fix bugs in the Sun Windows l&f that are not addressed by the other project. Hopefully, we can move the most important solutions to the Sun 1.5.0 Windows l&f and catch up with each others project.
