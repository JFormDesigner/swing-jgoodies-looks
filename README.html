JGoodies Looks, Version 1.2 Preview 1

The JGoodies look&feels make your Swing applications and applets look better. They have been optimized for readability, precise micro-design and usability.

The Looks require Java 1.4 or later.

Getting Started

First, download and run the JGoodies Looks Demo. Then follow the Quick Start and read the user's guide. If you want to use a JGoodies l&f in NetBeans read here.

Distribution Contents

docs/api   JavaDocs directory
docs/guide   User's Guide
 
examples/lib   library for the simple Looks demo
examples/tiny.jar   precompiled Tiny example
examples/demo.jar   precompiled simple Looks demo
 
src/examples/tiny   sources for the tiny example
src/examples/demo   sources for the simple Looks demo
src/photoshop   PhotoShop files for Plastic icons
 
LICENSE.txt   the BSD license agreement
README.html   this readme file
RELEASE-NOTES.txt   Release Notes
build.xml   ANT build file
default.properties   ANT default build properties
 
looks-1.2pre1-src.zip   zipped sources for all JGoodies l&fs
looks-1.2pre1.jar   a JAR containing all JGoodies l&fs
looks-win-1.2pre1.jar   a JAR containing the JGoodies Windows l&f
plastic-1.2pre1.jar   a JAR containing the JGoodies Plastic l&fs
