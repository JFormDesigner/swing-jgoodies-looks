An Introduction to the JGoodies Looks Demo

The JGoodies Looks Demo utilizes, presents, and tests the JGoodies Windows Look&Feel and the JGoodies Plastic Look&Feel.

The demo consists of two parts: in the launcher you can select the look, color theme, and define options provided by the looks; a sample application then shows the effect of the current settings.
