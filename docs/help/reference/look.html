Look and Theme

Setting a JGoodies Look

This is the preferred way, if you are allowed to import third party libraries: 
   try {
      UIManager.setLookAndFeel(new Plastic3DLookAndFeel());
   } catch (Exception e) {
   }

Setting a JGoodies Look by Name

You can set a JGoodies look using its class name: 
   try {
      UIManager.setLookAndFeel("com.jgoodies.plaf.windows.ExtWindowsLookAndFeel");
   } catch (Exception e) {
   }

Setting a Plastic Theme

Plastic and Plastic3D choose a default theme that should be appropriate for the given operating system. You can set a theme via: 
   PlasticLookAndFeel.setMyCurrentTheme(new DesertBlue());
