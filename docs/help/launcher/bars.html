Menu Bar and Tool Bar

The JGoodies Looks provide optional settings that let you tweak the appearance of menu bars and tool bars. These options are intended for bars that are located in an application's header. They are switched off by default.

For details, see the reference.

Header Style

You can tell the JGoodies looks, that a menu bar or tool bar is in the header by setting the header style. The JGoodies looks will use this hint to choose look-specific special borders and 3D effects. This is the recommended option.

Border Style

You can override the header style by setting a look-specific border style: empty, etched, or separator.

Plastic 3D Hint

Plastic chooses an appropriate default pseudo 3D effect for the specified header and border style. You can override the default by setting a Plastic 3D hint.
