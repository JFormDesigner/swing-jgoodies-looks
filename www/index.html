Free High-Fidelity Windows and Multi-Platform Appearance

The JGoodies look&feels make your Swing applications and applets look better. They have been optimized for readability, precise micro-design and usability. And they simplify the multi-platform support by using similar widget dimensions. Anyway, many people have reviewed them as elegant.

   

Download

You can download the latest release, archived versions, and the precompiled JGoodies Looks Demo. Each Looks release includes the binary library JAR, library sources, documentation, and tutorial sources.

Documentation

To get started open the file README.html, browse the HTML documentation, then launch the JGoodies Looks Demo and browse the tutorial sources that ship with the Looks distribution. A step-by-step description helps you learn how to make your applications look better using the JGoodies Looks. You may also study the presentation "First Aid for Swing" at the JGoodies articles page.

Support

Do not send personal mail!

You may subscribe to the users mailing list and browse or search the list's archive. And you can direct new questions to users@looks.dev.java.net. Please register with this list before you post; otherwise I need to manually accept your messages. Also, you can use the issue tracker to find issues regarding the Looks. As a project observer you can log defects, request features and suggest enhancements.

Before you enter a bug report, please make sure that the problem appears with the JGoodies L&f only, not with the parent Sun L&f (Plastic -> Metal, JGoodies Windows -> Sun Windows). If you have found a bug, please provide the following information: JRE vendor, JRE version, JRE build, OS, OS version, Looks version, Look&feel used. For example: Sun J2SE 1.4.2_05, Windows 2000, Looks 1.3, PlasticXP.

Features

JGoodies Windows L&F :: Precise Emulation
The JGoodies Windows look&feel focuses on a precise emulation on Windows 95/98/NT/ME/2000 in the following areas: menus, icons, colors, borders, fonts, font sizes, insets, and widget dimensions. It honors the screen resolution (96dpi vs. 120 dpi) to adjust sizes, insets, and widget dimensions. The Windows XP emulation still lags behind the classic emulation, nevertheless it has been improved significantly in the Looks version 1.2.

High-Fidelity Multi Platform Appearance
Plastic is an elegant multi-platform look&feel family that works very well on all Windows platforms, including XP. The Plastic themes have been tweaked for a harmonic integration with the majority of desktops so that your applications won't alienate a Windows user.

Simplifying Multi Platform Support
The looks help you support different Java versions and operating systems. Each look is consistent over the Java versions 1.4.0x, 1.4.1x, 1.4.2x, and 1.5 - to the possible extent w. r. t. to features of the underlying Swing API.

In addition, Plastic, the JGoodies Windows look&feel, and the Mac OS X Aqua look use similar widget dimensions and widget font baselines to minimize layout differences between these looks.
